# **App Name**: NexBox TV

## Core Features:

- Live Channel Playback: Automatically play the first channel in fullscreen upon load, using a YouTube embed for seamless streaming.
- Sequential Channel Navigation: Allow users to switch to the next live channel in a predefined list via a 'Next' button or a dedicated keyboard shortcut, looping back to the first channel.
- Interactive Overlay Toggle: An 'OK' button and corresponding keyboard shortcut to toggle a semi-transparent bottom navigation overlay for channel browsing and information.
- Dynamic Channel Overlay Display: When active, the overlay displays the current channel's name, its number (e.g., 1/6), and allows direct selection of other channels from the list.
- Contextual News Brief Tool: An AI tool that provides a brief, AI-generated summary of a relevant top news story or current affairs, linked contextually to the currently playing news channel.
- Persistent Channel Data: Manages the static list of YouTube live stream URLs and channel names, mapping them for application use.

## Style Guidelines:

- Primary accent color: A deep, digital blue (#267EF2) chosen to evoke a modern, reliable, and engaging 'screen-like' experience against a dark background. (HSL: 220, 70%, 50%)
- Background color: A very dark, desaturated blue-gray (#121417) to provide a sleek, minimalist 'set-top box' aesthetic, akin to a TV's ambient display. (HSL: 220, 15%, 8%)
- Secondary accent color: A bright, clear cyan (#85E3F2) used sparingly for interactive elements, highlights, and status indicators, providing visual contrast and drawing attention to key actions. (HSL: 190, 50%, 70%)
- All text uses 'Inter', a grotesque sans-serif font known for its clarity and neutrality, aligning with a modern, objective television interface feel.
- Simple, high-contrast, and generously sized icons, optimized for legibility and interaction from a distance, mimicking remote-control usability.
- A full-screen video player is paramount. Navigation elements are either discreetly hidden or presented as large, minimal controls in an automatically receding semi-transparent overlay at the bottom of the screen.
- Subtle, smooth transition effects are used when switching channels and toggling the navigation overlay, providing a fluid and premium 'TV-like' user experience.
